# Santa Claus Classification Bot

Этот бот для Telegram классифицирует изображения и определяет, находится ли на них Санта Клаус.

## Как работает бот

1. **Команда /start**: При отправке этой команды бот приветствует пользователя и объясняет, как использовать его.

2. **Отправка изображения**: Пользователь может отправить изображение любого размера и формата боту.

3. **Классификация изображения**: Бот загружает изображение и передает его на обработку нейронной сети, обученной определять, содержится ли на изображении Санта Клаус или нет.

4. **Ответ бота**: После анализа изображения бот отправляет сообщение с результатом классификации - "Santa Claus" (если на изображении обнаружен Санта Клаус) или "Not Santa Claus" (если не обнаружен).
